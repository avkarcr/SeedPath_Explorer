# SeedPath Explorer
Утилита для работы с BIP-39 сид-фразами и генерации Ethereum-адресов с поддержкой различных путей деривации. Проект предназначен для офлайн-анализа сид-фраз, восстановления адресов и перебора корректных комбинаций слов с валидной checksum.

## Структура проекта
- main.py — точка входа
- base/bip44_hdwallet.py — работа с HD-кошельками по стандарту BIP-44
- base/legacy.py — генерация адресов по legacy-пути Ethereum
- base/combinations.py — перебор и проверка комбинаций сид-фраз

## Принцип работы
В файле seeds.txt построчно задаются сид-фразы.

Модуль main.py читает конфигурацию и, в зависимости от значения переменной LEGACY_MODE (config.py), выбирает способ деривации адресов:
- legacy-режим (старые Ethereum-пути)
- BIP-44 HD-режим

Сгенерированные адреса сохраняются в файл addresses.txt.

## Модуль combinations
Модуль combinations предназначен для работы с одной сид-фразой.
Он перебирает все возможные перестановки слов (для 12 слов — около 479 миллионов комбинаций) и отбирает только те варианты, которые имеют корректную BIP-39 checksum.
Результатом является список валидных сид-фраз, пригодных для дальнейшего использования в деривации адресов.
